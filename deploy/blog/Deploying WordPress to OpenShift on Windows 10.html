Deploying WordPress to OpenShift on Windows 10

Posted: Sat, 01 Jan 2011
  • wordpress
  • openshift

Install Container Development Kit

Before you start, you need to register a OpenShift account. Then Download Red Hat Container Development Kit.Please follow the install instructions: https://developers.redhat.com/products/cdk/hello-world/#fndtn-windows

Now I had downloaded cdk-3.5.0-1-minishift-windows-amd64.exe. Spin up the console and type the command: cdk-3.5.0-1-minishift-windows-amd64.exe setup-cdk Setting VirtualBox as the default hypervisor with the following command: cdk-3.5.0-1-minishift-windows-amd64 config set vm-driver virtualbox Check the setting with: cdk-3.5.0-1-minishift-windows-amd64 config get vm-driver

Permanently set Red Hat Developer username and password

setx MINISHIFT_USERNAME [username] setx MINISHIFT_PASSWORD [password]

This command will start a Red Hat Linux VM: cdk-3.5.0-1-minishift-windows-amd64 start
